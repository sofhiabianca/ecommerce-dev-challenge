# E-commerce Landing Page

This project is a simple e-commerce landing page built with Angular and SCSS.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sofhiabianca/ecommerce-dev-challenge.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    ng serve
    ```
    Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Project Structure

The project is structured as follows:

-   `src/app/core`: Core components like the header and footer.
-   `src/app/shared`: Shared components, services, and models.
-   `src/app/features`: Feature components for the landing page.

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.io/cli) page.
